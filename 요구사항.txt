<1>총 관리자는 호텔의 정보를 확인 하기를 원한다.
   호텔 -  호텔이름 / 대표자 / 호텔주소 /  호텔연락처 / 호텔 우편번호 / 개업일자
   
   
<2>예약은 예약순서에 따라 고유한 예약번호를 부여받고 해당 고객에 대한 정보를 확인 할 수 있으며, 한 고객이 여러 예약을 할 수 있다.
   예약 - 예약번호/ 고객번호 / 체크인 / 체크아웃 / 조식 / 인원수 / 예약시 날짜 / 객실번호


<3>호텔은 방문한 고객들 역시 고유한 고객번호를 발급하여 이전 방문(예약) 이력을 조회할 수 있기를 원한다.  
   고객 - 고객번호/ 고객이름/ 핸드폰번호 / 고객주소 / 호텔이름


<4>각각의 객실은 객실 등급에 따라 가격이 정해진다.
   객실 - 객실번호/ 객실등급 / 뷰코드 / 객실금액


<5>결제 완료된 금액은 일일 매출액과 매출 총액에 반영된다.
   결제 - 예약번호 / 결제수단 / 결제금액 / 고객번호


<6>호텔은 발렛을 위해 고객들의 예약 시 차량 정보만을 조회하길 원한다.
   차량 - 차량번호 / 예약번호 / 고객번호 


<7>호텔은 총 예약횟수 , 총 매출액을 한 눈에 보길 원한다. 
   매출 -  총 예약횟수 / 총 매출액
